# HIG Changelog

## 1.0.0 — 2026-03-22 (iOS 18 / macOS 15 / iOS 26 Liquid Glass)

- Initial data created from Apple HIG documentation
- Sources: developer.apple.com/design/human-interface-guidelines/, Flutter Cupertino source, Apple Design Resources Figma
- Covers: 9 system colors, 6 grays, 4 labels, 4 fills, 6 backgrounds, separators
- Typography: iOS 13 styles + macOS 9 styles + Dynamic Type scale
- Components: Navigation, Controls, Content, Feedback, Input
- Patterns: Animation (spring presets), Gestures, Layout, Interaction
- Platforms: iOS, macOS, watchOS, visionOS, Web
- Stacks: SwiftUI, React, React Native, HTML/Tailwind (Tier 1)
- Includes Liquid Glass (iOS 26) preliminary data
